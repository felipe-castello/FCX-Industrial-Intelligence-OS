# fcx-agent-skills-adapter

Adapter FCX para `felipe-castello/agent-skills`.

## Objetivo

Executar skills FCX para agentes tecnicos, industriais, eletronicos e quant.

## Feature flag

`ENABLE_AGENT_SKILLS=false`

## Fallback

Quando desabilitado ou indisponivel, retorna `fallback` e nao bloqueia o FCX Core.
