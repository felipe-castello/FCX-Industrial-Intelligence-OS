# FCX Quant Trading App

Aplicação wrapper para o módulo FCX Quant Intelligence powered by TradingAgents.

## Objetivo

Expor uma camada controlada de pesquisa e simulação para análise quantitativa, sem execução de ordens reais.

## Modo inicial

```text
research
```

## Submódulos FCX Quant Intelligence

- Market Scanner.
- Technical Analyst.
- Sentiment Analyst.
- News Analyst.
- Risk Manager.
- Portfolio Manager.
- Decision Logger.
