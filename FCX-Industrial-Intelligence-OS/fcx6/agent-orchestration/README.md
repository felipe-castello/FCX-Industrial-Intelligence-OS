# Agent Orchestration

## Objetivo

Orquestrar fluxos multiagente para diagnóstico, resposta a alarmes, análise energética e apoio à manutenção.

## Tecnologias previstas

- LangGraph para grafos de decisão.
- Redis para estado temporário.
- PostgreSQL para estado persistente e auditoria.
- NestJS para APIs de orquestração.
- Filas para execução assíncrona.

## Próximas tarefas

- Definir grafos iniciais.
- Mapear estados e transições.
- Definir pontos de aprovação humana.
- Criar modelo de auditoria.
- Planejar limites de segurança para ações automáticas.
