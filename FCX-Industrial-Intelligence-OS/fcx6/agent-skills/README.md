# Agent Skills

## Objetivo

Criar uma camada de habilidades reutilizáveis para agentes industriais, permitindo diagnósticos, análises e recomendações com contexto operacional do FCX.

## Tecnologias previstas

- NestJS para APIs internas.
- TypeScript para contratos de skills.
- LangGraph para fluxos de execução.
- LlamaIndex ou Haystack para contexto documental.
- PostgreSQL para auditoria de execuções.
- Redis para filas e cache de contexto.

## Próximas tarefas

- Definir contrato padrão de skill.
- Criar catálogo inicial de skills industriais.
- Mapear permissões por perfil de usuário.
- Definir formato de entrada, saída e evidências.
- Planejar auditoria de execução de agentes.
