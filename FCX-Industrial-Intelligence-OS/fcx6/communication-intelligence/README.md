# Communication Intelligence

## Objetivo

Transformar conversas operacionais em informação estruturada, vinculando mensagens, incidentes, ativos, alarmes e ordens de serviço.

## Tecnologias previstas

- WhatsApp Cloud API.
- Nango para integrações de comunicação e produtividade.
- NestJS para ingestão e classificação.
- LlamaIndex para contexto conversacional.
- PostgreSQL para histórico e vínculos.

## Próximas tarefas

- Definir canais suportados.
- Criar taxonomia de mensagens.
- Planejar ingestão de WhatsApp, e-mail e chamados.
- Definir regras de privacidade e retenção.
- Mapear vínculo entre conversas e ativos industriais.
