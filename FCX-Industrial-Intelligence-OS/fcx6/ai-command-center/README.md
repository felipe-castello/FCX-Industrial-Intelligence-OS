# AI Command Center

## Objetivo

Criar uma interface central para operadores, gestores e engenharia interagirem com agentes, alertas, recomendações e conhecimento do FCX.

## Tecnologias previstas

- React para interface.
- LibreChat como referência ou base de interface conversacional.
- Backend NestJS para APIs de comando.
- WebSocket ou SSE para atualizações em tempo real.
- JWT/RBAC para controle de acesso.

## Próximas tarefas

- Definir personas de uso.
- Mapear comandos principais por perfil.
- Projetar tela de chat operacional.
- Definir integração com Agent Skills.
- Planejar histórico auditável de conversas.
