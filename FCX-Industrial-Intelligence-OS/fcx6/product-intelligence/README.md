# Product Intelligence

## Objetivo

Medir uso, valor, adoção e impacto operacional do FCX como produto SaaS industrial.

## Tecnologias previstas

- PostgreSQL para eventos de produto.
- TimescaleDB para séries temporais de uso.
- Grafana para painéis internos.
- Prometheus para métricas técnicas.
- Backend NestJS para coleta e agregação.

## Próximas tarefas

- Definir eventos de produto.
- Criar métricas de adoção por cliente.
- Mapear jornadas críticas do usuário.
- Definir KPIs de valor operacional.
- Planejar dashboards internos de produto.
