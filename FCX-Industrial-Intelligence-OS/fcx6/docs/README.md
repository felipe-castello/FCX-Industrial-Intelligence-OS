Centralizar documentação técnica, arquitetura, decisões, ADRs, riscos, planos de migração e contratos da evolução FCX 6.0.

- Mermaid para diagramas.
- ADRs para decisões arquiteturais.

- Diagramas C4 para arquitetura.
