# Decision Intelligence

## Objetivo

Apoiar decisões operacionais e executivas com recomendações explicáveis baseadas em risco, custo, SLA, energia e saúde dos ativos.

## Tecnologias previstas

- NestJS para APIs de recomendação.
- PostgreSQL/TimescaleDB para dados operacionais.
- LangGraph para fluxos de decisão.
- Modelos preditivos para risco e tendência.
- Grafana e frontend FCX para visualização.

## Próximas tarefas

- Definir modelo de recomendação.
- Mapear critérios de priorização.
- Criar matriz de risco operacional.
- Definir trilha de decisão e aprovação.
- Planejar comparação entre recomendação e resultado real.
