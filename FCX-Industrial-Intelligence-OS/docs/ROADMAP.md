# Roadmap MVP

- Subir Docker Compose: PostgreSQL/TimescaleDB, Redis, EMQX, Grafana.
- Criar backend NestJS com módulos base.
- Criar frontend React institucional + dashboard inicial.

- Alarmes por limite técnico.

- Dashboards Grafana.
- Relatórios técnicos.
- Integrações Boss/Sitrad/ThingsBoard.

## Semana 4
- Agentes IA: vibração, energia, refrigeração e manutenção.
- Deploy em VPS Hostinger.
